Чанки

  • Файлы чанков находятся в папке шаблонов + подпапка chunks
  • Чанки можно использовать в шаблонах, для переиспользования и чистоты кода в шаблонах
  • Чанк в чанке использовать нельзя
  • Чанки являются обычными blade шаблонами и принимают данные в виде массива как это делается через @include('templateName') в blade шаблонизаторе
  • Все принятые данные в чанке (принимаются в виде массива), будут преобразованы в переменные
  • Окончание .blade.php в названиях файлов чанков - обязательно
  • Все поддерживаемые переменные в чанке, должны быть переданы при вызове самого чанка

Пример

Вызов чанка в шаблоне. Название файла чанка будет camel_entity_name.blade.php 

{{ $chunk->get('camel_entity_name', ['var1' => 'text 1 for chunk', 'var2' => 'text 2 for chunk']) }}

Содержание чанка camel_entity_name.blade.php

Var 1: {{ $var1 }}, Var 2: {{ $var2 }}

Результат генерации чанка 

Var 1: text 1 for chunk, Var 2: text 2 for chunk
